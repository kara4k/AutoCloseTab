AutoCloseTabs

AutoCloseTab - extension that allows to automatically close tabs where URL address or text on the page matches the specified patterns. As a template, you can specify both plain text (in this case, it is checked whether URL or text of the page contains it) or regular expressions. Each template is specified with a new line, without special delimiters. Blank lines are allowed between patterns to visually separate particularly long expressions. When closing a tab by page text, slight delays are possible, since to determine if text matches, it's necessary to wait until the main content of the page is loaded (not images). Good service for testing regular expressions: regex101.com.
